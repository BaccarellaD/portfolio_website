Starship Theremin

Unity   Leap SDK   C#

What It Is

Starship Theremin was a hackathon project from SBU Hacks 2018. It was built by a team of 3 programmers and 1 3D artist over the course of 24 Hours. It is an infinite scrolling retro style space shooter, with some awesome controls and visuals.

Traditionally, flying games tend to be hard to control. You need to be able to change spatial positioning along with full rotation. With new computer input technology we felt like this didn't have to be the case. We used leap motion hand controls to map positional and rotational data onto the players spaceship. This allows anyone to pick up and play the game without a steep learning curve for the controls. Beyond that it just feels awesome to play.

What I Learned

The project was a great exercise in time management. Having such a short window to work in helped us focus on the core mechanics of the game and really try to polish those as much as possible. We used unity version control extensively though out the project
