Starship Theremin

Unity ⚬ C# ⚬ Leap Motion

Starship Theremin was made during a 24 hour hackathon at Stony Brook University.

It is an infinite scrolling, 80's style space shooter that uses leap motion controls for hand tracking. The position and rotation are mapped onto the ship allowing for natural feeling flight controls using only your hands.

Metro Map Maker

Java ⚬ JavaFX ⚬ JSON

This was the final project for my CSE 219 class.

It's a metro map editor that allows users to create subway style map. It has a shortest path finder between any two stations that handles cases where there could be looping paths. Maps are able to be saved on disk through a custom format based on json or exported to image file.
